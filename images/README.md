## Cara Menggunakan:
1. Download atau ambil foto-foto sesuai nama dan ukuran di atas
2. Simpan semua foto di folder `images/`
3. Pastikan nama file sesuai dengan yang tercantum di atas
4. Website akan otomatis menampilkan foto-foto tersebut

## Tips Foto:
- Gunakan foto berkualitas tinggi (minimal 72 DPI untuk web)
- Pastikan pencahayaan foto bagus dan tidak terlalu gelap
- Foto produk (menu) sebaiknya dengan background yang bersih
- Foto interior/suasana café bisa lebih candid dan natural
